
## dayone-to-evernote
Import Day One Classic journals to Evernote on MacOS
## Install
```
npm install -g dayone-to-evernote
```
## Usage
```
dayone-to-evernote ~/Dropbox/Journal_dayone
```

## Force fully resync
```
rm -r ~/Dropbox/Journal_dayone/.dayone-to-evernote
dayone-to-evernote ~/Dropbox/Journal_dayone
```
